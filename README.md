# Zodiac — pop astrology (multilingua)

Sito statico, multilingua (IT · EN · FR · ES · PT-BR). Nessun backend.
Entrata: **index.html** (hub da cui raggiungi tutte le pagine).

Pagine: Zodiac Zine (segni), Tema Natale, Oroscopo del Giorno, Match del Giorno, Componi la Frase.
Cartelle: `lang/` (i18n), `coppie/`, `natal/` (motori di calcolo).

## Pubblicare (GitHub Pages)
1. Crea un repo pubblico su GitHub.
2. Carica tutti questi file (mantieni le cartelle).
3. Settings → Pages → Deploy from a branch → `main` / `(root)` → Save.
Online in ~1 min su `https://UTENTE.github.io/REPO/`.

In alternativa: trascina la cartella su Netlify o Vercel (HTTPS automatico).
